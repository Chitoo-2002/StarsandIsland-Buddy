# 星砂岛小助手

这是一个农场经营辅助工具，旨在通过严谨的数学模型，帮你打破种地时的选择困难症，精准定位当前阶段最赚钱的作物与发展路线。

1. **下载运行**：在 Releases 页面下载最新的 `StarSandCalc.exe` 应用程序，直接双击运行即可，无需安装环境。
2. **导入存档（可选）**：如果你之前有备份过数据，前往【设置】页面，点击“打开存档路径”，将 `StarSandData.json` 放入弹出的文件夹中。
3. **刷新同步**：回到程序主页面，点击顶部的“刷新数据”按钮，你的所有自定义作物、肥料和设置将瞬间就位。

---

## 📂 文件结构说明

* `StarSandBuddy.exe`：应用程序本体。
* `StarSandData.json`：核心存档文件（运行程序后自动生成）。包含所有的作物属性、你自定义的肥料列表、系统设置参数以及你个人的界面排版数据。**请妥善保管或定期备份此文件！**
