# 星砂岛作物收益计算器

这是一个农场经营辅助工具，旨在通过严谨的数学模型，帮你打破种地时的选择困难症，精准定位当前阶段最赚钱的作物与发展路线。


## 🌟 核心功能亮点

### 1. 全流派收益全景精算
不再纠结于“直接卖原果”还是“花时间做果酱”。系统会针对每种作物，自动铺开所有加工流派（直售、一级加工、高级加工等）在搭配不同肥料时的表现。
* **预计时薪**：实时计算扣除种子与化肥成本后的每小时纯收入。
* **单次纯收益**：直观展示单次收割到底能把多少金币落入腰包。

### 2. 肥料性价比对比实验室
商店里的化肥到底是投资还是智商税？在这里一测便知。
* **增量直观对比**：清晰展示施肥后相比于不施肥，时薪到底提升了多少、单次利润增加了多少。

### 3. 可自定义的数据库
你的农场规则，完全由你定义。
* **自由调配**：不论是新版本增加了新作物、新肥料，还是你想测试某种假想化肥的收益，都可以直接在程序中快速实现修改，系统会自动重新构建所有算法公式。

### 4. 个性化记忆与无缝存档
* **界面记忆**：你在表格上拖拽的任何一处列宽，系统都会永久记住，确保每次打开都是你最顺手的布局。
* **极简迁移**：所有配置与数据均保存在一个独立的本地文件中，重装系统或换电脑时，只需拖入存档，一键点击“刷新数据”，瞬间恢复原貌。

---

## 🚀 快速上手指南

1. **下载运行**：在 Releases 页面下载最新的 `StarSandCalc.exe` 应用程序，直接双击运行即可，无需安装环境。
2. **导入存档（可选）**：如果你之前有备份过数据，前往【设置】页面，点击“打开存档路径”，将 `StarSandData.json` 放入弹出的文件夹中。
3. **刷新同步**：回到程序主页面，点击顶部的“刷新数据”按钮，你的所有自定义作物、肥料和设置将瞬间就位。

---

## 📂 文件结构说明

* `StarSandCalc.exe`：应用程序本体。
* `StarSandData.json`：核心存档文件（运行程序后自动生成）。包含所有的作物属性、你自定义的肥料列表、系统设置参数以及你个人的界面排版数据。**请妥善保管或定期备份此文件！**
